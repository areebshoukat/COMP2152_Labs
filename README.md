# COMP2152 — Open Source Development with Python (Labs)

Lab starter files and exercises for COMP2152, Winter 2026 at George Brown College.

## How to Use

1. Fork this repo to your own GitHub account
2. Clone your fork
3. For each lab, create a branch (e.g., `lab_week10`), do your work there, then merge into `master`
4. Submit your GitHub repo link on D2L — no zip files, no screenshots

## Starter Files

From Week 05 onward, each lab has starter files with `# TODO` markers. Open the starter, find the TODOs, and fill them in. The provided code (helpers, main blocks) should not be modified.
